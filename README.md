# 双轨人生 (Dual Track Life)

一个帮助用户探索和规划人生可能性的智能助手应用。

## 项目简介

双轨人生是一个创新的生活规划和人生探索平台，它通过AI技术帮助用户：

- 探索人生的多种可能性
- 获取个性化的人生建议
- 记录和分析重要生活事件
- 进行深度的自我反思
- 规划未来的发展轨迹

## 技术栈

### 后端
- Spring Boot
- Spring AI
- DashScope AI
- H2 Database
- JPA

### 前端
- Vue.js 3
- TypeScript
- Tailwind CSS

## 主要功能

- 🤖 AI生命导师：提供个性化的人生建议和指导
- 📝 事件记录：记录和分析重要的人生事件
- 🔄 平行宇宙：探索不同的人生可能性
- 📊 轨迹分析：分析个人成长轨迹和发展趋势
- 💭 深度反思：辅助进行有意义的自我反思

## 快速开始

### 环境要求
- Java 17+
- Node.js 16+
- Maven 3.6+

### 后端启动
```bash
cd backend
./mvnw spring-boot:run
```

### 前端启动
```bash
cd frontend
npm install
npm run dev
```

## 环境变量配置

后端需要配置以下环境变量：
- DASHSCOPE_API_KEY：阿里云通义千问API密钥
